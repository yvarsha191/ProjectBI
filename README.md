Acted as a Business Intelligence Analyst for AdventureWorks Cycles, a fictional manufacturing company to transform raw data into professional-quality reports and dashboards to track KPIs, compare regional performance, analyze product-level trends, and identify high-value customers.

Extracted, transformed, and load data using Power Query, build AdventureWorks data model using Star & snowflake schemas, Bi-directional filters, and Model layouts, introduced data analysis expressions (DAX), created calculated columns and measures, explored row and filter context, by  applying powerful tools like filter functions, iterators, and time intelligence patterns.

Finally, build visually creative and interactive reports by formatting basic charts, and add interactivity with bookmarks, slicer panels, parameters, tooltips, report navigation.
